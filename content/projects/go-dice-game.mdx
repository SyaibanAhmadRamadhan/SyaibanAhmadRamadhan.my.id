title: Golang Dice Game
description: dice game using Golang, the game will stop if only one player has the dice. If the player rolls the dice and gets a score of 6 then the number of the player's dice will be deducted, and if the player rolls the dice and gets a score of 1 then the player will get +1 point. Players will not be able to roll the dice if their dice total is 0. The player with the most points wins.
